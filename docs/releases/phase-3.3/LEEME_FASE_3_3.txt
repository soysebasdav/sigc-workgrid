NOTAS IMPORTANTES
- La simulación de retención NO elimina información.
- La verificación automática de backup comprueba metadatos, manifiesto, checksum registrado y existencia del objeto en Storage. No sustituye una restauración completa de prueba.
- La salud de Render, red externa o infraestructura de Supabase no se consulta mediante APIs del proveedor en esta fase; el panel se basa en telemetría interna de Orkesta.
- La configuración regional prepara i18n, pero no traduce automáticamente todos los textos existentes.
